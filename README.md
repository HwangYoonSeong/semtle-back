# Semtle project

1. 작성한 라우터 파일은 router라는 폴더
2. 작성한 함수 파일은 js라는 폴더
3. 스키마들은 schemas 폴더
4. 작성한 미들웨어는 router/middlewares -> 또는 그냥 올려도 제가 바꾸겠습니당.
## set up
```
npm install
```

## 실행
```
node app.js
```
